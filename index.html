Recovery Options: System Won't Boot At All

When your Windows 11 system fails to start completely, try these recovery options in the recommended order:

First Option

Startup Repair No Internet Required

Automatically diagnoses and fixes issues that prevent Windows from starting.

  • Boot to Recovery Environment by forcing shutdown during startup 3 times
  • Select "Troubleshoot" > "Advanced options" > "Startup Repair"
  • Wait for the diagnostic and repair process to complete
  • Restart your computer when prompted
i This option preserves all user data and applications.

Startup Repair addresses these common issues:

  • Missing or corrupted boot configuration data
  • Corrupted system files needed during boot
  • Incompatible drivers preventing boot

If Startup Repair runs multiple times without success, it's an indication that you should try the next recovery option.

Second Option

Command Prompt Recovery No Internet Required

Use command-line tools to manually repair boot issues.

  • Boot to Recovery Environment
  • Select "Troubleshoot" > "Advanced options" > "Command Prompt"
  • Run boot repair commands (see command list below)
  • Type "exit" and restart your computer
bootrec /scanos
bootrec /rebuildbcd
bootrec /fixmbr
bootrec /fixboot
bcdboot C:\Windows /s C:
i This option preserves user data but requires technical knowledge.
Third Option

Reset PC (Keep My Files) No Internet Required

Reinstalls Windows 11 while preserving your personal files.

  • Boot to Recovery Environment
  • Select "Troubleshoot" > "Reset this PC"
  • Choose "Keep my files"
  • Select "Local reinstall" (not Cloud download)
  • Follow the on-screen instructions to complete
i This option preserves personal files but removes applications and settings.

Recovery Options: System Boots But Crashes

When your Windows 11 system starts occasionally but crashes or freezes, try these recovery options:

First Option

Safe Mode No Internet Required

Boot Windows with minimal drivers to identify problematic software.

  • Press F8 during startup (or from WinRE, select "Troubleshoot" > "Advanced options" > "Startup Settings")
  • Select option 4 or F4 for Safe Mode
  • Once in Safe Mode, uninstall recently added programs or drivers
  • Run System File Checker: Open Command Prompt as admin and type "sfc /scannow"
i This option preserves all user data and allows troubleshooting.
Second Option

System Restore No Internet Required

Revert system files to a previous working state.

  • From Safe Mode or WinRE, open System Restore
  • Select a restore point from before the problem started
  • Follow the wizard to complete the restoration
  • Restart your computer to apply changes
i This option preserves personal files but reverts system changes.
Third Option

DISM and System File Repairs Internet Recommended

Repair Windows system image and files.

  • Boot to Safe Mode with Networking
  • Run Command Prompt as administrator
  • Run DISM: "DISM /Online /Cleanup-Image /RestoreHealth"
  • Run SFC: "sfc /scannow"
  • Restart and test system stability
DISM /Online /Cleanup-Image /RestoreHealth
sfc /scannow
i This option preserves all data while repairing system files.

Recovery Options: System Infected With Malware

When your Windows 11 system is compromised by malware, try these recovery options:

First Option

Windows Defender Offline Scan No Internet Required

Run a comprehensive malware scan outside of normal Windows operation.

  • Boot to Recovery Environment
  • Select "Troubleshoot" > "Advanced options" > "Command Prompt"
  • Type "MpCmdRun.exe -SignatureUpdate" to update definitions (if internet available)
  • Type "MpCmdRun.exe -Scan -ScanType 3 -Boot" to run a full offline scan
  • Let the scan complete and remove any threats found
i This option preserves all legitimate user data while removing malware.

Recovery Options: New Hardware or Hard Drive

When installing Windows 11 on new hardware or after a hard drive failure, use these options based on available hardware:

First Option (with DVD Drive)

DVD Installation Media Internet Required (for creation)

Install Windows 11 using an official installation DVD.

  • On another computer, download Windows 11 ISO from Microsoft
  • Burn ISO to DVD using Windows built-in burning tool or third-party software
  • Insert DVD into the new/repaired computer and boot from it
  • Follow the installation wizard to install Windows 11
  • Activate Windows with your product key or Microsoft account
i This is a clean installation on new hardware. No previous data will be preserved.

Required: Working DVD drive, Windows 11 product key

Hardware check: Modern DVD drives support booting from DVD. Enable DVD boot in BIOS/UEFI.

First Option (without DVD Drive)

USB Installation Media Internet Required (for creation)

Install Windows 11 using a bootable USB drive.

  • On another computer, download Media Creation Tool from Microsoft
  • Run the tool and select "Create installation media for another PC"
  • Choose USB flash drive option (8GB+ recommended)
  • Insert the USB drive into the new/repaired computer and boot from it
  • Follow the installation wizard to install Windows 11
i This is a clean installation on new hardware. No previous data will be preserved.

Required: USB port, Windows 11 product key

Hardware check: Enable USB boot in BIOS/UEFI (typically F2, F10, F12, or Del during startup).

Second Option (Enterprise)

Network Installation (PXE Boot) No Internet Required (Local Network)

Install Windows 11 from a network deployment server.

  • Ensure PXE boot server is configured on your network (typically by IT)
  • Connect new/repaired computer to the network via Ethernet
  • Enter BIOS/UEFI and enable network boot (PXE)
  • Boot the computer and press the network boot key (often F12)
  • Select Windows 11 deployment image from the network menu
i This is a clean installation on new hardware. No previous data will be preserved.

Required: Network adapter, access to PXE server

Enterprise only: This option requires a configured deployment server (WDS, MDT, or SCCM).

Third Option

Recovery Drive from Another PC No Internet Required

Use a recovery drive created on another Windows 11 PC.

  • On another Windows 11 PC, create a recovery drive: Start > type "Create a recovery drive"
  • Follow the wizard to create the recovery drive on a USB (8GB+ recommended)
  • Insert the recovery drive into the new/repaired computer
  • Boot from the USB and select "Recover from a drive"
  • Follow the wizard to install a clean copy of Windows 11
i This is a clean installation. No previous data will be preserved.

Required: USB port, access to another Windows 11 PC

Note: This creates a generic Windows 11 installation without specific drivers or applications.

Alternative Option

Windows 11 Installation without TPM/Secure Boot Internet Required (for download)

Install Windows 11 on hardware that doesn't meet official requirements.

  • Create standard Windows 11 installation media (USB/DVD)
  • Press Shift+F10 during installation to open Command Prompt
  • Type "regedit" and navigate to: HKEY_LOCAL_MACHINE\SYSTEM\Setup
  • Create a new key called "LabConfig"
  • Create DWORD values: "BypassTPMCheck"=1, "BypassSecureBootCheck"=1, "BypassRAMCheck"=1
  • Close Registry Editor and continue installation
i Use this only when hardware doesn't meet Windows 11 requirements.

Warning: This method installs Windows 11 on unsupported hardware. You may not receive updates or security patches in the future. Use at your own risk.

Alternative registry bypass commands (via Command Prompt during setup):

reg add HKLM\SYSTEM\Setup\LabConfig /v BypassTPMCheck /t REG_DWORD /d 1
reg add HKLM\SYSTEM\Setup\LabConfig /v BypassSecureBootCheck /t REG_DWORD /d 1
reg add HKLM\SYSTEM\Setup\LabConfig /v BypassRAMCheck /t REG_DWORD /d 1
